# Form Field "Key-Value to JSON" to Voyager BREAD 
Hook

## Installation

You can use the artisan command below to install this hook

```bash
php artisan hook:install key-value-json
```

### Required
```bash
"laravel/framework": "5.7.*"
"tcg/voyager": "^1.1"
```
